# Your Projects

This is where your writing lives. Each project gets its own folder.

## Starting a new project

Click the **+** button in the "Project Files" panel (in the sidebar), or use the command palette: **Creative Writer: New Writing Project**.

## Adding an existing project

Already have a manuscript? Drop it in here:

1. Create a folder inside `projects/` with your project name
   (for example: `projects/my-novel/`)
2. Put your files inside it -- any `.md` or `.txt` files will work
3. You can organize with subfolders like `chapters/`, `notes/`, `characters/`
4. The AI assistant will automatically read your project structure

### Suggested layout for an existing novel

```
projects/
  my-novel/
    outline.md           (optional — your plot outline)
    style-guide.md       (optional — your writing voice, or let AI generate it)
    chapters/
      01-chapter-one.md
      02-chapter-two.md
    characters/
      protagonist.md
      antagonist.md
    notes/
      ideas.md
    research/
      sources.md
    images/
```

You don't need all of these. Even a single folder with one `.md` file works. The AI adapts to whatever structure you use.
